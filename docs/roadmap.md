# Roadmap

This roadmap is non-binding. Items listed here are not part of v0.1.0 unless implemented, tested, and documented.

## v0.1.x

- Stabilize record mapping and reports.
- Improve examples from user feedback.
- Keep production dependencies at zero.
- Improve error metadata where real usage exposes gaps.

## Future Candidates

- Support simple JavaBeans/field-based classes.
- Multiple profiles.
- Watch/reload support.
- Optional validation module.
- YAML/TOML/JSON as separate opt-in modules, not core dependencies.
- Conversion SPI.
- Framework adapters.
- Cloud secret manager adapters.

## Out Of Scope For v0.1.0

- Hot reload.
- YAML/TOML/JSON in core.
- Bean Validation dependency.
- Spring, Quarkus, Jakarta, or Lombok integration.
- Cloud secret manager integration.
- Logging framework dependency.
