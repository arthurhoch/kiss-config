# GitHub Pages

The Pages workflow publishes:

- Markdown files from `docs/`
- generated Javadocs under `javadocs/`

Manual setup:

1. In GitHub repository settings, enable Pages.
2. Select GitHub Actions as the Pages source.
3. Push to `main` or run the Pages workflow.

The workflow uses official GitHub Pages actions and does not require a separate documentation generator.
